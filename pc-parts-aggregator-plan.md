# 🖥️ План разработки агрегатора цен на комплектующие ПК

## 💡 Концепция проекта

**Цель:** сайт который находит самую дешёвую цену на комплектующие ПК со всех площадок России — новые и б/у — в одном месте.

**Уникальность:** единственный агрегатор который объединяет специализированные магазины (DNS, Ситилинк) + маркетплейсы (Ozon, WB) + б/у рынок (Авито) с умным матчингом товаров.

**Парсинг цен — законен** при соблюдении правил:

- ✅ Собираем только публичные данные (цены, названия, характеристики)

- ✅ Не перегружаем серверы (задержки между запросами)

## 📡 Источники данных

### Фаза 1 — Простые (без серьёзной защиты)
- **DNS** (dns-shop.ru) — простой HTML
- **Ситилинк** (citilink.ru) — простой HTML
- **Регард** (regard.ru) — простой HTML

### Фаза 2 — Маркетплейсы (умеренная защита)
- **Ozon** — headless браузер, ротация User-Agent
- **Wildberries** — аналогично, лимит 1 req/мин
- **Яндекс Маркет** — умеренная защита

### Фаза 3 — Б/у рынок (сложная защита)
- **Авито** — отдельная архитектура, резидентные прокси, частая переиндексация

---

## 🛡️ Защита от блокировок

### Стратегии (от дешёвых к дорогим)

1. **Ротация User-Agent** — бесплатно, базовая защита
2. **Случайные задержки** — 2-20 сек между запросами, бесплатно
3. **Экспоненциальный backoff** — при получении 429/403
4. **Прокси дата-центр** — для DNS/Ситилинк, ~$5-10/мес
5. **Прокси резидентные** — для Ozon/WB, ~$25-40/мес (5 GB)
6. **Прокси мобильные** — для Авито, ~$10-20/мес
7. **Playwright Stealth** — скрытие признаков автоматизации

### Бюджет на прокси

| Источник | Тип прокси | Стоимость/мес |
|---|---|---|
| DNS, Ситилинк, Регард | Дата-центр | ~$5-10 |
| Ozon, Wildberries | Резидентные | ~$25-40 |
| Авито | Резидентные/мобильные | ~$15-25 |
| **Итого** | | **~$45-75/мес** |

## 🧠 Матчинг товаров (ключевая задача)

### Проблема
Один товар называется по-разному на разных площадках:
- DNS: `NVIDIA GeForce RTX 4070 SUPER MSI GAMING X SLIM 12G`
- WB: `Видеокарта MSI RTX 4070 Super 12gb Gaming`
- Авито: `4070 super msi почти новая`

### Решение — 3 уровня матчинга

**Уровень 1 — Структурированные атрибуты** (для спец. магазинов)
- Категория + бренд + модель + объём памяти
- Покрывает ~80% новых товаров

**Уровень 2 — LLM нормализация** (для маркетплейсов и Авито)
```
Вход: "видеокарта 4070 super msi почти новая"
Выход: { category: "GPU", brand: "MSI", model: "RTX 4070 Super", memory_gb: 12, condition: "used" }
```
Стоимость: копейки (Gemini Flash)

**Уровень 3 — Векторное сходство** (сложные случаи)
- pgvector + эмбеддинги названий
- Для поиска дубликатов и похожих товаров

---

## 🗄️ Структура базы данных

```
categories          — CPU, GPU, RAM, SSD, Motherboard и т.д.
    ↓
products            — уникальные товары после нормализации
    ↓
offers              — предложения с каждого источника (цена, ссылка, наличие)
    ↓
price_history       — история цен для графиков
    ↓
users               — пользователи (для уведомлений)
    ↓
price_alerts        — подписки на снижение цены
```

### Итоговый результат для пользователя
```
RTX 4070 Super
├── Новый от 43 500₽ (Ozon) 
├── Новый от 45 990₽ (DNS)
├── Новый от 47 200₽ (Ситилинк)
└── Б/у от 28 000₽ (Авито) — экономия 36%
```

## ⚠️ Ключевые риски

| Риск | Вероятность | Как снизить |
|---|---|---|
| WB/Ozon меняют структуру сайта | Высокая | Мониторинг + быстрые фиксы, абстракция парсеров |
| Авито блокирует IP | Высокая | Резидентные прокси, лимиты запросов |
| Плохой матчинг товаров | Средняя | LLM + ручная модерация первых данных |
| Медленный рост трафика | Высокая | SEO с первого дня, контент-маркетинг |
| Юридические претензии | Низкая | Соблюдать robots.txt, не копировать фото |
